## 今日完成

1. **enum 落地**：`internal/domain/enum/enum.go`
   - `VehicleStatus`：draft / published / unpublished / deleted（与 PRD 状态机对齐）
   - `PriceMode`：show_price / phone_inquiry

2. **model 落地**：`internal/domain/model/vehicle.go`
   - `Vehicle` 结构体：ID、CategoryID、Name、CoverMediaID、PriceMode、MSRPPrice、Status、SellingPoints、SortOrder、CreatedAt、UpdatedAt
   - `Publish()`、`Unpublish()` 方法：状态变更 + UpdatedAt 更新

3. **rule 落地**：`internal/domain/rule/vehicle_publish.go`
   - `PublishRequirements` 结构体：HasCoverImage、HasDetailImages、HasRequiredParams
   - `CanPublishVehicle(v, requirements)`：纯函数，校验草稿状态、必填项、封面图、详情图、参数完整性

4. **vehicles handler**：`internal/transport/http/handler/vehicles.go`
   - VehiclesListHandler、VehiclesCreateHandler、VehiclesUpdateHandler、VehiclesDeleteHandler（占位实现）

5. **router 挂载**：`internal/transport/http/router/router.go`
   - `admin/vehicles`：GET /、POST /、PUT /:vehicle_id、DELETE /:vehicle_id
   - `public/vehicles`：GET /（复用 VehiclesListHandler）
